Product Introduction

We've built a complete closed-loop system from emotion recognition to music generation, making music creation simple, fast, and emotionally rich

How to Use Emo2Music

Users simply upload a facial photo or use the camera to capture expressions in real-time. The system generates personalized music matching the current emotion within 3 minutes.

Real-time Emotion Recognition

Based on ResEmoteNet model with 94.76% accuracy (RAF-DB dataset), supporting 7 basic emotion classifications.

Intelligent Prompt Generation

Utilizing LLM to transform emotion features into structured MusicGen Prompts, automatically matching music style, BPM, and instrument combinations.

Personalized Music Generation

Generating high-quality music based on AudioCraft framework, designing exclusive styles for different emotions with 85% emotion-matching accuracy.
